Changelog

  • 19. März 2006:
    Rudimentäres Administrationsinterface zu Demonstrationszwecken
    Unter folgender Adresse zu erreichen: mr-magister.pytalhost.de/admin/list.php
    Wenn Sie auf die Links klicken, erreichen Sie den WYSIWYG-Rich-Text Editor, der eine Word-ähnliche Oberfläche besitzt
  • 22. März 2006:
    Editor Einstellungen wurden geändert. Dieser startet nun immer mit einer Größe von 800 * 600 Pixeln. Des Weiteren sind zwei neue Funktionen aktiviert worden, sodass nun einfacher Wordtext eingefügt werden kann. Sehen Sie dazu die zwei neuen Button in 2. Zeile 4. und 5. von links.
  • 27. März 2006:
    Der Editor hat sich um eine neue Funktion erweitert: Wenn Sie nun einen Link einfügen, brauchen Sie die Adresse nicht mehr einzugeben. Makieren Sie den Text und klicken Sie auf das Link Symbol (die beiden Kettengleider). In dem nun erscheinenden Dialogfeld klicken Sie auf den Button rechts neben der Eingabebox für die Adresse. Dort können Sie ein Verzeichniss, bzw. demnächst auch einen Artikel auswählen. Somit hat sich die interne Verlinkung vereinfacht. Verbesserungsvorschläge?
  • 3. April 2006:
    Funktion vom 27. März verbessert und debuggt. Nun wird der Link korrekt aus dem Auswahlfenster übernommen.
  • 7. April 2006:
    Aus programmiertechnischen Gründen ist der Link vom 19. März in folgenden geändert worden: mr-magister.pytalhost.de/admin/old/list.php
  • 20. April 2006:
    Von nun an kann aus den Suchergebnissen heraus auch der entsprechende Artikel aufgerufen werden.

Roadmap

  • Artikelanzeige:
    Über dem Artikel soll eine Anzeige erscheinen, wo man sich gerade auf der Seite befindet. Beispiel: home > grammatik > partizipien
  • Admin:
    Einbindung des Admin-Interfaces in das Grundgerüst der Seite, sowie die Weiterentwicklung.
  • Es wird eine Fehlerverarbeitungsroutine mit eingebaut die bei auftretenden Fehlern eine Mail an den Entwickler schickt.
